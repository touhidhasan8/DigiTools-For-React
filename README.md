
#  DigiTools - Tech Service Platform

## Overview
DigiTools is a modern web application that provides a wide range of tech-related services. Users can easily select a package that suits their needs, choose multiple services, view the price of each service, and see the total cost before placing their order.

---

##  Features

  1. Select different service packages.
2. Choose multiple services.
  3. View selected services in a list.
 4. See individual service prices.
  5. Automatic total cost calculation.
 6. User-friendly and responsive UI.
 7. Smooth interaction using React Hooks.


## Tech Stack

This project is built using:

   React
   Tailwind CSS
   DaisyUI
  React Hooks (useState etc...)

The application is fully responsive and works smoothly across:
